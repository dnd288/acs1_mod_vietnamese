## USAGE NOTES

### Capitalization Rules

1. **Realm names:** Capitalize (e.g., Trúc Cơ, Kim Đan)
2. **Common terms:** Lowercase (e.g., tu luyện, linh khí)
3. **Proper nouns:** Capitalize (e.g., Côn Lôn, Thái Nhất Môn)

### Translation Guidelines

1. **Preserve Chinese names** for:
   - Sect names (e.g., 昆仑 → Côn Lôn, not "Kunlun Mountain")
   - Technique names (keep Chinese + translate meaning)
   - Location names (transliterate + translate if needed)

2. **Translate descriptively** for:
   - Common items (e.g., 灵石 → linh thạch)
   - Actions (e.g., 修炼 → tu luyện)
   - Concepts (e.g., 境界 → cảnh giới)

3. **Use established terms** from Vietnamese cultivation novels
